Bài 2: Quản lý các luồng nhập xuất (I/O Streams)

Java I/O Streams File Handling

Dữ liệu trong mạng di chuyển dưới dạng "luồng". Bài blog này sẽ đi sâu vào cách Java xử lý các dòng chảy dữ liệu này.

Phân loại luồng

Theo loại dữ liệu

  • Luồng byte (Byte stream): Xử lý dữ liệu nhị phân, sử dụng InputStream và OutputStream
  • Luồng ký tự (Character stream): Xử lý văn bản, sử dụng Reader và Writer

Theo thao tác

  • Luồng nhập (Input/Reader): Đọc dữ liệu từ nguồn
  • Luồng xuất (Output/Writer): Ghi dữ liệu ra đích

Luồng Byte

Sử dụng InputStream và OutputStream để xử lý dữ liệu nhị phân:

import java.io.*;

// Đọc file sử dụng FileInputStream
try (FileInputStream fis = new FileInputStream("data.bin")) {
    int byteData;
    while ((byteData = fis.read()) != -1) {
        // Xử lý từng byte
        System.out.print((char) byteData);
    }
} catch (IOException e) {
    e.printStackTrace();
}

// Ghi file sử dụng FileOutputStream
try (FileOutputStream fos = new FileOutputStream("output.bin")) {
    byte[] data = "Hello World".getBytes();
    fos.write(data);
} catch (IOException e) {
    e.printStackTrace();
}

Luồng Ký tự

Sử dụng Reader và Writer để xử lý văn bản:

import java.io.*;

// Đọc file văn bản
try (FileReader fr = new FileReader("input.txt")) {
    int charData;
    while ((charData = fr.read()) != -1) {
        System.out.print((char) charData);
    }
} catch (IOException e) {
    e.printStackTrace();
}

// Ghi file văn bản
try (FileWriter fw = new FileWriter("output.txt")) {
    fw.write("Hello from Java!");
} catch (IOException e) {
    e.printStackTrace();
}

Kỹ thuật vùng đệm (Buffering)

Sử dụng các lớp như BufferedReader, BufferedOutputStream để tăng tốc độ đọc/ghi bằng cách giảm số lần thao tác trực tiếp với thiết bị:

import java.io.*;

// BufferedReader để đọc hiệu quả hơn
try (BufferedReader br = new BufferedReader(
        new FileReader("largefile.txt"))) {
    String line;
    while ((line = br.readLine()) != null) {
        System.out.println(line);
    }
} catch (IOException e) {
    e.printStackTrace();
}

// BufferedWriter để ghi hiệu quả hơn
try (BufferedWriter bw = new BufferedWriter(
        new FileWriter("output.txt"))) {
    bw.write("Line 1");
    bw.newLine();
    bw.write("Line 2");
} catch (IOException e) {
    e.printStackTrace();
}

Lớp hỗ trợ định dạng

Scanner

Scanner dùng để đọc dữ liệu đầu vào (như bàn phím) và phân tích các kiểu dữ liệu khác nhau:

import java.util.Scanner;

Scanner scanner = new Scanner(System.in);

System.out.print("Nhập tên: ");
String name = scanner.nextLine();

System.out.print("Nhập tuổi: ");
int age = scanner.nextInt();

System.out.print("Nhập điểm: ");
double score = scanner.nextDouble();

scanner.close();

PrintWriter

PrintWriter dùng để ghi dữ liệu định dạng ra đầu ra:

import java.io.*;

try (PrintWriter pw = new PrintWriter(
        new FileWriter("formatted.txt"))) {
    pw.printf("Tên: %s%n", "Nguyễn Văn A");
    pw.printf("Tuổi: %d%n", 25);
    pw.printf("Điểm: %.2f%n", 8.5);
} catch (IOException e) {
    e.printStackTrace();
}

Try-with-Resources

Java 7+ cung cấp try-with-resources để tự động đóng các stream, đảm bảo không bị rò rỉ tài nguyên:

// Tự động đóng stream khi kết thúc block
try (FileInputStream fis = new FileInputStream("file.txt")) {
    // Sử dụng stream
} // Stream tự động được đóng ở đây

Best Practices

  • Luôn sử dụng try-with-resources để đảm bảo stream được đóng đúng cách
  • Sử dụng buffered streams cho các file lớn để tăng hiệu suất
  • Chọn đúng loại stream (byte vs character) dựa trên loại dữ liệu
  • Xử lý exceptions một cách phù hợp
