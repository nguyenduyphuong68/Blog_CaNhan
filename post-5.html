Bài 5: Lập trình Socket cho giao thức TCP

Java Socket TCP Client-Server

Hướng dẫn xây dựng mô hình Khách/Chủ (Client/Server) sử dụng kết nối tin cậy TCP.

Mô hình Khách/Chủ

  • Server: Cung cấp dịch vụ và lắng nghe các yêu cầu từ client
  • Client: Gửi yêu cầu và đợi đáp trả từ server

Socket

Socket cho phép lập trình viên xem liên kết mạng như một luồng dữ liệu để đọc/ghi, che giấu các chi tiết mức thấp của giao thức mạng. Socket là điểm cuối của kết nối hai chiều giữa hai chương trình chạy trên mạng.

Quy trình cho Client

  1. Tạo Socket với IP/Port của Server
  2. Thiết lập luồng nhập/xuất
  3. Trao đổi dữ liệu
  4. Đóng kết nối

Ví dụ Client

import java.net.Socket;
import java.io.*;
import java.net.UnknownHostException;

public class TCPClient {
    public static void main(String[] args) {
        String serverHost = "localhost";
        int serverPort = 12345;
        
        try (Socket socket = new Socket(serverHost, serverPort);
             PrintWriter out = new PrintWriter(
                 socket.getOutputStream(), true);
             BufferedReader in = new BufferedReader(
                 new InputStreamReader(socket.getInputStream()))) {
            
            // Gửi dữ liệu đến server
            out.println("Hello Server!");
            
            // Đọc phản hồi từ server
            String response = in.readLine();
            System.out.println("Server response: " + response);
            
        } catch (UnknownHostException e) {
            System.err.println("Không tìm thấy host: " + serverHost);
        } catch (IOException e) {
            System.err.println("Lỗi I/O: " + e.getMessage());
        }
    }
}

Quy trình cho Server

  1. Tạo ServerSocket
  2. Lắng nghe yêu cầu bằng accept() (trả về một Socket khi có kết nối)
  3. Xử lý dữ liệu
  4. Đóng kết nối

Ví dụ Server đơn giản

import java.net.ServerSocket;
import java.net.Socket;
import java.io.*;

public class TCPServer {
    public static void main(String[] args) {
        int port = 12345;
        
        try (ServerSocket serverSocket = new ServerSocket(port)) {
            System.out.println("Server đang lắng nghe trên port " + port);
            
            // Chấp nhận kết nối từ client
            Socket clientSocket = serverSocket.accept();
            System.out.println("Client đã kết nối: " + 
                clientSocket.getInetAddress());
            
            try (BufferedReader in = new BufferedReader(
                    new InputStreamReader(clientSocket.getInputStream()));
                 PrintWriter out = new PrintWriter(
                     clientSocket.getOutputStream(), true)) {
                
                // Đọc dữ liệu từ client
                String message = in.readLine();
                System.out.println("Nhận từ client: " + message);
                
                // Gửi phản hồi
                out.println("Hello Client! Server đã nhận: " + message);
            }
            
            clientSocket.close();
            
        } catch (IOException e) {
            System.err.println("Lỗi server: " + e.getMessage());
        }
    }
}

Server xử lý nhiều Client

Server đơn giản chỉ xử lý một client tại một thời điểm. Để xử lý nhiều client đồng thời, cần sử dụng multithreading:

import java.net.ServerSocket;
import java.net.Socket;
import java.io.*;
import java.util.concurrent.ExecutorService;
import java.util.concurrent.Executors;

public class MultiClientTCPServer {
    private static final int PORT = 12345;
    private static final int MAX_CLIENTS = 10;
    
    public static void main(String[] args) {
        ExecutorService executor = Executors.newFixedThreadPool(MAX_CLIENTS);
        
        try (ServerSocket serverSocket = new ServerSocket(PORT)) {
            System.out.println("Server đang lắng nghe trên port " + PORT);
            
            while (true) {
                Socket clientSocket = serverSocket.accept();
                System.out.println("Client mới kết nối: " + 
                    clientSocket.getInetAddress());
                
                // Xử lý mỗi client trong một thread riêng
                executor.submit(new ClientHandler(clientSocket));
            }
        } catch (IOException e) {
            System.err.println("Lỗi server: " + e.getMessage());
        }
    }
    
    static class ClientHandler implements Runnable {
        private Socket clientSocket;
        
        public ClientHandler(Socket socket) {
            this.clientSocket = socket;
        }
        
        @Override
        public void run() {
            try (BufferedReader in = new BufferedReader(
                    new InputStreamReader(clientSocket.getInputStream()));
                 PrintWriter out = new PrintWriter(
                     clientSocket.getOutputStream(), true)) {
                
                String message;
                while ((message = in.readLine()) != null) {
                    System.out.println("Nhận từ client: " + message);
                    out.println("Echo: " + message);
                }
            } catch (IOException e) {
                System.err.println("Lỗi xử lý client: " + e.getMessage());
            } finally {
                try {
                    clientSocket.close();
                } catch (IOException e) {
                    e.printStackTrace();
                }
            }
        }
    }
}

Các phương thức quan trọng

Socket

  • Socket(String host, int port): Tạo socket và kết nối đến server
  • getInputStream(): Lấy luồng nhập để đọc dữ liệu
  • getOutputStream(): Lấy luồng xuất để ghi dữ liệu
  • close(): Đóng kết nối

ServerSocket

  • ServerSocket(int port): Tạo server socket lắng nghe trên port
  • accept(): Chấp nhận kết nối từ client, trả về Socket
  • close(): Đóng server socket

Best Practices

  • Luôn sử dụng try-with-resources hoặc đảm bảo đóng socket đúng cách
  • Sử dụng buffered streams để tăng hiệu suất
  • Xử lý exceptions một cách phù hợp
  • Sử dụng thread pool cho server xử lý nhiều client
  • Đặt timeout cho socket để tránh treo vô hạn
  • Xử lý graceful shutdown cho server
